# I-Whalle Backend API

Express.js server with TypeScript, PostgreSQL, and Firebase integration.

## 📚 Documentation

- [Setup Guide](SETUP.md)
- [API Reference](API.md)
- [Database Schema](../backend/prisma/schema.prisma)

## 🚀 Quick Start

```bash
npm install
npm run dev
```

API runs at http://localhost:5000

## 📝 Environment Variables

Create `.env` from `.env.example`:

```bash
cp .env.example .env
```

Edit with your Firebase and database credentials.

## 🧪 Testing

```bash
npm test
npm run test:watch
```

## 📦 Build & Deploy

```bash
npm run build
npm start
```
