# I-Whalle Frontend

React + TypeScript + Vite admin dashboard.

## 📚 Documentation

- [Setup Guide](SETUP.md)
- [Components](docs/COMPONENTS.md)

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Frontend runs at http://localhost:3000

## 📝 Environment Variables

Create `.env` from `.env.example`:

```bash
cp .env.example .env
```

Configure with your Firebase credentials and API URL.

## 🏗️ Build

```bash
npm run build
npm run preview
```

## 🧪 Code Quality

```bash
npm run lint
npm run format
npm run type-check
```
